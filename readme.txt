Display video icon of your choice next to Posts tagged 'video'.

== Description ==

This plugin creates a new Recent Posts widget that lets you display a video icon of your choice next to Posts tagged &#8217;video&#8217;. You can also specify the post title, number of posts, order (descending or ascending), and whether or not to display the post date.

== Installation ==

1. Extract the contents of the .zip archive
2. Upload the recent-posts-video-icon folder to your wp-content/plugins directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to Appearance --> Widgets to add the widget to any widgetized areas.
5. Upload enclosed video_icon.gif, or your own icon file, via FTP or Media Library and specify in Widget properties.
6. Add the tag 'video' to Posts that you want the icon to appear next to.
